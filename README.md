# AI Objects Linux

A collection of Linux-native AI tools for 3D object generation and processing.
Managed under WSL2 (Ubuntu 24.04) with NVIDIA GPU support.

## Projects

### [Trellis2](./Trellis2/)
TRELLIS.2 — Native and Compact Structured Latents for 3D Generation (Microsoft Research)
- Model: Image  3D Mesh (GLB + PBR textures)
- Env: conda activate trellis2
- Requirements: NVIDIA GPU 24GB+ VRAM (tested on A100/H100)

## System Requirements
- OS: Ubuntu 24.04 LTS (WSL2)
- GPU: NVIDIA with CUDA 12.4
- Conda: Miniconda3
